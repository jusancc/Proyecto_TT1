Detailed Description

Declaración de la función que interpola parámetros EOP del IERS.

Esta función proporciona valores interpolados o discretos de parámetros de orientación de la Tierra (EOP) necesarios para modelos de tiempo y transformaciones de marcos.

Recupera o interpola los parámetros IERS para una fecha UTC dada.

Los parámetros incluyen desplazamientos del polo, diferencias de tiempo y correcciones de nutación para una fecha en tiempo UTC.

Recupera o interpola los parámetros IERS para una fecha UTC dada.

Dependiendo del parámetro interp, esta función realiza interpolación lineal o devuelve los valores directos para el día UTC dado. La información extraída incluye desplazamientos polares, UT1-UTC, LOD, correcciones de nutación y diferencia TAI-UTC.

Parameters
eopMatriz de datos EOP ya cargada (13 x N columnas).
Mjd_UTCFecha en tiempo UTC (Modified Julian Date).
interpModo de interpolación: 'l' = lineal, 'n' = sin interpolación.
Returns
Tupla con:
  • x_pole [rad]
  • y_pole [rad]
  • UT1 - UTC [s]
  • LOD [s]
  • Δψ [rad] (nutación en longitud)
  • Δε [rad] (nutación en oblicuidad)
  • dx_pole [rad]
  • dy_pole [rad]
  • TAI - UTC [s]
Exceptions
Terminael programa si la fecha `Mjd_UTC` no se encuentra en los datos.
