double gmst ( double Mjd_UT1)

Calcula el Tiempo Sidéreo Medio de Greenwich (GMST) en radianes.

< Funciones matemáticas estándar

Esta función sigue la fórmula del IAU 1982 para calcular el GMST, que representa el ángulo entre el meridiano de Greenwich y el punto Aries, medido en el plano ecuatorial.

Parameters
Mjd_UT1Tiempo universal UT1 (Modified Julian Date).
Returns
GMST en radianes, normalizado al intervalo [0, 2π).
