Detailed Description

Implementación de la matriz de transformación LTC (Local Tangent Coordinate).

Esta función construye la matriz que transforma coordenadas cartesianas ECEF al sistema de coordenadas locales topocéntricas tangentes a la superficie terrestre en una ubicación específica dada por latitud y longitud.
