Detailed Description

Declaración de la función que calcula el azimut, la elevación y los vectores de la base local (topocéntrica) a partir de un vector posición.

Esta función devuelve los ángulos de observación (azimut y elevación) así como los vectores unitarios del sistema local (sur, este, cenit) para observaciones terrestres.

Calcula el azimut, la elevación y los vectores base (topocéntricos) locales.

A partir de un vector de posición s en coordenadas cartesianas, esta función devuelve:

  • El azimut (medido desde el norte hacia el este).
  • La elevación (ángulo sobre el horizonte).
  • El vector derivada parcial del azimut respecto a s.
  • El vector derivada parcial de la elevación respecto a s.
