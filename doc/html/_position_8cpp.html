Detailed Description

Implementación de la función que convierte coordenadas geodésicas a cartesianas ECEF.

Esta función transforma una ubicación dada por longitud, latitud y altitud (geodésicas) en coordenadas cartesianas del sistema ECEF (Earth-Centered, Earth-Fixed), utilizando el modelo elipsoidal del WGS84.
