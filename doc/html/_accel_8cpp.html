Detailed Description

Implementación de la función que calcula la aceleración total que actúa sobre un satélite, considerando perturbaciones armónicas y gravitatorias de cuerpos celestes usando efemérides JPL DE430.

Function Documentation

◆ accel()

Matrix & accel ( double x,
Matrix & Y )

Calcula la aceleración total sobre un cuerpo a partir de su estado y tiempo.

< Efemérides JPL DE430

Esta función computa las fuerzas que actúan sobre un satélite, incluyendo:

  • Aceleración por armónicos esféricos del campo gravitacional terrestre.
  • Atracción gravitatoria del Sol, la Luna y planetas (opcional según configuración).

El sistema de referencia se transforma aplicando matrices de precesión, nutación, movimiento del polo y ángulo horario de Greenwich.

Parameters
xTiempo transcurrido (en segundos) desde la época de referencia (MJD_UTC).
YVector de estado (6x1) con posición y velocidad cartesianas [km, km/s].
Returns
Referencia a un vector de derivadas del estado (6x1): [vx,vy,vz,ax,ay,az].
