Calcula el ángulo entre dos vectores tridimensionales.

< Clase Matrix personalizada para operaciones vectoriales

Utiliza el producto escalar para determinar el ángulo entre dos vectores dados. En caso de que uno de los vectores tenga magnitud muy pequeña, se devuelve un valor indefinido (999999.1) como marcador de error.

Parameters
vec1Primer vector (3x1 o 1x3).
vec2Segundo vector (3x1 o 1x3).
Returns
Ángulo entre vec1 y vec2 en radianes. Si no se puede calcular, retorna 999999.1.
