Detailed Description

Prototipo de la función que convierte coordenadas geodésicas a posición en coordenadas cartesianas.

Esta función transforma las coordenadas geodésicas (longitud, latitud, altitud) a coordenadas cartesianas ECEF (Earth-Centered, Earth-Fixed) usando el modelo elipsoidal de la Tierra.
