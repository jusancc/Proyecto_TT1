Calcula numéricamente el gradiente de la aceleración armónica ∂a/∂r.

< Aceleración por armónicos esféricos

Esta función evalúa la derivada parcial de la aceleración producida por el campo gravitacional armónico respecto a la posición usando diferencias finitas centradas. Se perturba cada componente de r una cantidad d/2 y se calcula la diferencia entre aceleraciones.

Parameters
rVector de posición (3x1).
UMatriz de transformación (3x3) del sistema inercial al cuerpo.
n_maxOrden máximo del modelo armónico.
m_maxGrado máximo del modelo armónico.
Returns
Referencia a una matriz (3x3) con el gradiente ∂a/∂r.
Note
Esta aproximación es numérica y puede ser costosa para modelos grandes.
