Detailed Description

Implementación de la función R_y para generar una matriz de rotación sobre el eje Y.

Esta función construye una matriz 3x3 de rotación alrededor del eje Y para un ángulo especificado en radianes. Se utiliza comúnmente en transformaciones espaciales dentro de la mecánica orbital y sistemas de referencia tridimensionales.
