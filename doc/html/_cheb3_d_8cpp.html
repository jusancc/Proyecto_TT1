Evalúa una serie de interpolación de Chebyshev en 3D.

Esta función calcula la posición 3D en el tiempo t mediante una serie de Chebyshev de segundo orden. Usa el método de Clenshaw para evaluar los polinomios y aplica coeficientes para las tres componentes espaciales (X, Y, Z).

Parameters
tTiempo de evaluación.
NNúmero de coeficientes (orden del polinomio + 1).
TaInicio del intervalo de interpolación.
TbFin del intervalo de interpolación.
CxCoeficientes Chebyshev para la coordenada X.
CyCoeficientes Chebyshev para la coordenada Y.
CzCoeficientes Chebyshev para la coordenada Z.
Returns
Referencia a un vector (1x3) con la posición interpolada [x, y, z].
