Detailed Description

Declaración de la función que evalúa una serie de interpolación de Chebyshev en tres dimensiones.

Esta función evalúa una serie de polinomios de Chebyshev para obtener una posición 3D interpolada en el intervalo [Ta, Tb] a partir de coeficientes dados.

Esta función calcula la posición 3D en el tiempo t mediante una serie de Chebyshev de segundo orden. Usa el método de Clenshaw para evaluar los polinomios y aplica coeficientes para las tres componentes espaciales (X, Y, Z).
