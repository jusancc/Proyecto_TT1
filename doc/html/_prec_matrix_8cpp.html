Detailed Description

Implementación de la función que calcula la matriz de precesión.

Esta función genera la matriz de precesión entre dos fechas dadas, usando el modelo IAU 1976. La precesión es una rotación lenta del eje de la Tierra que cambia las coordenadas astronómicas a lo largo del tiempo.

Returns
Matriz de rotación 3x3 que representa la transformación debida a la precesión.

La matriz se calcula como Rz(-z) * Ry(theta) * Rz(-zeta), donde los ángulos se obtienen de las expresiones del modelo IAU 1976.
