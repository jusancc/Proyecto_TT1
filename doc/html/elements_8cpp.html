tuple< double, double, double, double, double, double, double > elements ( Matrix & y)

Calcula los elementos orbitales keplerianos a partir del vector de estado.

Esta función toma como entrada un vector de estado compuesto por posición y velocidad y devuelve una tupla con los elementos keplerianos: semilatus rectum, semi-eje mayor, excentricidad, inclinación, longitud del nodo ascendente, argumento del periastro y anomalía media.

Parameters
yVector de estado (6x1): posición (x,y,z) y velocidad (vx,vy,vz)
Returns
tuple<double, double, double, double, double, double, double> con:
  • p: semilatus rectum
  • a: semi-eje mayor
  • e: excentricidad
  • i: inclinación
  • Omega: longitud del nodo ascendente
  • omega: argumento del periastro
  • M: anomalía media
